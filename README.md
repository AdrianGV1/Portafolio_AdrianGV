## 🎯 Perfil Profesional Objetivo

Basado en el análisis de las 5 ofertas, el perfil profesional objetivo definido es:

> **Full Stack Web Developer con énfasis en Frontend, especializado en HTML5, CSS3, JavaScript y React, con conocimientos básicos de backend en Node.js y buenas prácticas de desarrollo web responsivo.**

Este perfil es el más demandado en el mercado actual, especialmente en startups y empresas de tecnología en crecimiento. Combina habilidades sólidas en el ecosistema JavaScript tanto en frontend como en backend básico, lo que permite manejar el ciclo completo de desarrollo de una aplicación web.

**Tecnologías clave:**
- React.js + JavaScript (frontend)
- HTML5 + CSS3 (estructura y estilos)
- Node.js básico (backend)
- Git + GitHub (control de versiones)
- Figma (diseño y prototipado)

---

## 🌟 Diferenciadores Clave

Se identificaron **2 diferenciadores** que distinguen a este perfil en el mercado:

**1. Dominio completo del ciclo de desarrollo web**
La capacidad de manejar tanto el frontend como nociones de backend permite asumir proyectos completos de forma autónoma, desde el diseño hasta el despliegue. Esto es especialmente valorado en equipos pequeños donde un desarrollador debe cubrir múltiples responsabilidades.

**2. Orientación al diseño y experiencia de usuario**
A diferencia de desarrolladores puramente técnicos, este perfil incorpora principios de diseño UI/UX y prototipado en Figma. Esta habilidad es altamente valorada en empresas que buscan desarrolladores capaces de tomar decisiones de diseño sin depender siempre de un diseñador dedicado.

## 🖥️ Justificación del Sitio Web

El sitio web desarrollado es un **portafolio profesional** orientado a la postulación a un puesto de Full Stack Web Developer Junior en una empresa de tecnología.

**Decisiones de diseño justificadas en la investigación:**
- **Sección de proyectos con cards visuales:** todas las ofertas mencionan la importancia de demostrar experiencia práctica con proyectos reales.
- **Sección de habilidades técnicas organizada por categorías:** refleja la estructura de evaluación técnica usada en los procesos de selección analizados.
- **Diseño dark mode:** alineado con las preferencias estéticas de la industria tech.
- **Sección de contacto con formulario funcional:** facilita la comunicación directa con reclutadores.
- **Diseño responsivo (mobile + desktop):** requerimiento explícito en las 5 ofertas analizadas.
- **Scroll snap por sección:** experiencia de navegación fluida y profesional.
